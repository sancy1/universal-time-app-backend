## 🚀 Quick Start

```bash
git clone <repository-url>

cd universal-time-app-backend

pnpm install

pnpm dev
```

Server runs on:

```text
http://localhost:3000
```

Swagger Documentation:

```text
http://localhost:3000/api-docs
```
